Why Is This Useful?

Have you ever wanted to send a link to a webpage to someone, but the section of the webpage that you are interested in is further down on the page? Rather than telling the person to scroll down the page until they reach the specific point that you want to show them, just send them a link that scrolls them to that section automatically!

Installation

Copy URL With Hash is an add-on for Mozilla Firefox. It is designed for both the desktop version of Firefox and the mobile (Android) version of Firefox! If you have a computer running Firefox or an Android device running Firefox, you can install Copy URL With Hash.

To install Copy URL With Hash, visit the add-on's page on Mozilla Add-ons.

For more information on using Copy URL With Hash, see the welcome page.
